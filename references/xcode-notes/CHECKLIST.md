# ✅ SIMPLE CHECKLIST

## Do These 4 Things in Xcode:

### 1. Delete Old File ❌
```
Find: Models.swift (in app target)
Right-click → Delete → Move to Trash
```

### 2. Add New File ✅
```
Right-click project → Add Files
Select: AutomationModels.swift
✓ Check "HomeKit Automator" target
Click: Add
```

### 3. Build 🔨
```
Press: ⌘B
Wait for: Build Succeeded
```

### 4. Run 🚀
```
Press: ⌘R
Look for: 🏠 icon in menu bar
Click: Icon → "Show Automations…"
```

---

## That's It! 🎉

If it works, you'll see:
- ✅ Menu bar icon
- ✅ New window opens
- ✅ Automation list (or empty state)
- ✅ Beautiful UI

If it doesn't work:
- Read `QUICK_START.md` for troubleshooting
- Most likely: forgot to add `AutomationModels.swift` to target
